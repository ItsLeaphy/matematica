Capítulo 1 — Trigonometria: resolução de triângulos
1. Triângulo retângulo — razões trigonométricas
sen α = op / hip
cos α = adj / hip
tan α = op / adj = sen α / cos α
op = cateto oposto · adj = cateto adjacente · hip = hipotenusa
Identidade fundamental: sen²α + cos²α = 1 (vale para qualquer ângulo)
Ideia chave: seno mede “altura relativa”, cosseno mede “base relativa”
Ângulos notáveis
ÂnguloSenCosTan
30°1/2√3/2√3/3
45°√2/2√2/21
60°√3/21/2√3

Macete do seno: 1/2 → √2/2 → √3/2 → 1
Cosseno: faz o caminho inverso
Insight ninja: tan α = sen α / cos α → se souber seno e cosseno, você já sabe tangente
Questões — pratique até virar reflexo

hip = 10 e op = 5

Nível 4
Sabendo que cos α = √3/2,
encontre sen α usando a identidade fundamental.
Dica de prova: se você travar, volte para o triângulo — desenhar resolve metade dos problemas
2. Ângulos complementares e suplementares
Complementares (somam 90°)
α + β = 90°

sen α = cos β
cos α = sen β
Ex: sen 30° = cos 60° = 1/2
Suplementares (somam 180°)
α + β = 180°

sen α = sen β
cos α = −cos β
Ex: sen 30° = sen 150°
cos 30° = −cos 150°
Leitura rápida: complemento troca seno ↔ cosseno
Suplementar: seno repete · cosseno inverte o sinal
Insight forte: isso funciona porque os ângulos estão “espelhados” no círculo trigonométrico
Dica de prova: sempre que ver ângulo estranho (tipo 150°, 120°, 135°), tenta transformar em 30°, 60°, 45°

3. Ângulos obtusos (90° < α < 180°)
Regra do seno
sen α = sen(180° − α)
Seno mantém o valor
Regra do cosseno
cos α = −cos(180° − α)
Cosseno troca o sinal
Seno: sempre positivo no 2º quadrante
Cosseno: sempre negativo no 2º quadrante
Tradução prática: transforma qualquer ângulo obtuso em um ângulo agudo equivalente
Exemplos clássicos
sen 120° = sen 60° = √3/2
cos 120° = −cos 60° = −1/2
sen 150° = sen 30° = 1/2
cos 150° = −cos 30° = −√3/2
sen 135° = sen 45° = √2/2
cos 135° = −cos 45° = −√2/2
Casos especiais
sen 90° = 1
cos 90° = 0
sen 180° = 0
cos 180° = −1

4. Simplificação de expressões
Estratégia: transformar ângulos e cancelar termos
Reduza tudo para ângulos notáveis ou use relações para eliminar termos
Ferramentas disponíveis
α > 90° → sen α = sen(180° − α)
soma = 90° → sen α = cos β
soma = 180° → sen mantém · cos troca sinal
Ideia central: não calcule direto — transforme primeiro
Exemplo guiado
sen 30° + sen 120° − sen 56° + sen 42°
sen 120° = sen 60° = √3/2
sen 56° = cos 34° (não notável)
42° não cancela diretamente → procurar pares
Conclusão: nem toda expressão simplifica totalmente — foque nos pares possíveis
Exemplo com cancelamento total
sen 20° − sen 160° + cos 44° + cos 136°
sen 160° = sen 20° → cancela
cos 136° = −cos 44° → cancela
Resultado final: 0
Regra de ouro: antes de calcular, procure cancelamentos — quase sempre existem

5. Lei dos senos
a / sen A = b / sen B = c / sen C = 2R
lados opostos aos respectivos ângulos · R = raio da circunferência circunscrita
Quando usar: 2 ângulos + 1 lado (ASA ou AAS)
Passo a passo
1. C = 180° − A − B
2. Use lado / seno do ângulo oposto
3. Monte proporção
4. Resolva a equação
Exemplo resolvido
A = 45°, B = 30°, a = 8 → achar b
a / sen A = b / sen B
8 / sen 45° = b / sen 30°
8 / (√2/2) = b / (1/2)
8√2 = b / (1/2) → b = 4√2
Dica: dividir por fração = multiplicar pelo inverso

6. Lei dos cossenos
a² = b² + c² − 2bc · cos A
Fórmula válida para qualquer triângulo (e análoga para os outros lados)
Quando usar: 2 lados + ângulo entre eles (SAS) ou 3 lados (SSS)
Para encontrar ângulo
cos A = (b² + c² − a²) / (2bc)
Uso: quando você tem os 3 lados e quer descobrir um ângulo
Conexão com Pitágoras
Se A = 90° → cos 90° = 0
então: a² = b² + c²
Pitágoras é um caso especial da lei dos cossenos
Insight: Lei dos cossenos = versão geral do Pitágoras
Cuidado: se o ângulo for obtuso, o cosseno é negativo (mas a fórmula já resolve isso)
Exemplo resolvido
b = 5, c = 7, A = 120° → achar a
a² = 5² + 7² − 2(5)(7) · cos 120°
a² = 25 + 49 − 70 · (−1/2)
a² = 74 + 35 = 109
a = √109
Questões — nível prova

Encontre a.
Nível 2

b = 6, c = 8, A = 60°
Encontre a.
Nível 3
Em um triângulo com lados:
a = 7, b = 5, c = 6
Calcule o ângulo A.
Nível 4
Calcule:
b = 10, c = 7, A = 120°
Encontre a.
Erro clássico: esquecer o sinal do cosseno (principalmente em ângulos obtusos)
Capítulo 2 — Conceitos trigonométricos básicos
7. Arcos e ângulos — conceitos fundamentais
Arco geométrico
Parte da circunferência entre dois pontos
Medida angular
Pode ser em graus ou radianos
Importante: medida angular ≠ comprimento do arco
Comprimento: depende do raio da circunferência
Ângulo: é uma medida “pura”, independe do tamanho da circunferência
Ângulo central
O ângulo central tem a mesma medida do arco que ele subtende
(mesma medida angular, não necessariamente mesmo comprimento)
Visualização: abrir o ângulo é como “puxar” o arco junto com ele
Erro comum: achar que arco maior sempre significa ângulo maior — depende do raio

8. Unidades de medida — graus e radianos
Grau
1° = 1/360 da circunferência
Radiano
1 rad = arco de comprimento igual ao raio
“esticando” o arco → tamanho = raio
Diferença chave: grau divide a volta · radiano mede diretamente o arco
Relação fundamental
180° = π rad
Toda conversão nasce daqui
Comprimento de arco
ℓ = α · r
α deve estar em radianos
Tradução: comprimento = ângulo × raio
Erro comum: usar grau na fórmula (sempre converta antes)
Insight forte: radiano conecta ângulo diretamente com comprimento

9. Conversão graus ↔ radianos
Regra geral
Grau → radiano
× (π / 180)
Radiano → grau
× (180 / π)
Base de tudo: 180° = π rad
Conversões simples (decorar)
30°45°60°90°
π/6π/4π/3π/2
120°150°180°270°360°
2π/35π/6π3π/22π
Conversões complexas
kπ/n = k × 180° / n
Substitua π por 180° e simplifique
Exemplos
17π/6 → 17 × 180 / 6 = 510°
= 360° + 150°
51π/10 → 51 × 180 / 10 = 918°
= 2 voltas + 198°
7π/4 → 7 × 180 / 4 = 315°
210° → 210 / 180 = 7/6 → 7π/6
Método: divide por 180 e simplifica, depois multiplica por π
Dica de cálculo: tenta dividir 180 primeiro — muitas contas ficam inteiras
Erro clássico: esquecer de simplificar fração

10. Circunferência orientada e trigonométrica
Circunferência orientada
Circunferência com sentido definido de percurso
Circunferência trigonométrica
Raio = 1 · centro (0,0)
ponto inicial A = (1,0)
Sentido positivo: anti-horário
Sentido positivo (+): anti-horário
Sentido negativo (−): horário
Ponto inicial: (1,0) → todos os ângulos partem daqui
Quadrantes
Quad Intervalo (graus) Intervalo (rad) Sen Cos
1º 0° a 90° 0 a π/2 + +
2º 90° a 180° π/2 a π + −
3º 180° a 270° π a 3π/2 − −
4º 270° a 360° 3π/2 a 2π − +
Padrão dos sinais: (+,+) → (+,−) → (−,−) → (−,+)
Erro clássico: esquecer em qual quadrante está o ângulo

11. Arcos côngruos (congruentes)
Dois arcos são côngruos quando terminam no mesmo ponto da circunferência
Podem dar voltas diferentes, mas chegam no mesmo lugar
Ideia central: diferença entre eles é múltiplo de uma volta completa
Expressões gerais
Graus
α + k · 360°
Radianos
x + 2kπ
k ∈ ℤ (inteiros positivos, negativos ou zero)
k positivo: voltas extras no sentido anti-horário
k negativo: voltas no sentido horário
Exemplos
Côngruos a 45°:
… −315°, 45°, 405°, 765° …
Côngruos a π/3:
… π/3 − 2π, π/3, π/3 + 2π …
Erro clássico: achar que ângulos diferentes sempre dão valores diferentes de seno/cosseno
Insight forte: arcos côngruos têm o mesmo seno e cosseno

12. Primeira determinação positiva
Menor arco não-negativo côngruo ao arco dado
Equivalente a “reduzir para a primeira volta”
Intervalo final: [0°, 360°)
Como calcular
Divide por 360° e pega o resto
Ou soma/subtrai 360° até entrar no intervalo
Exemplos
780° → 780 ÷ 360 = 2 (resto 60)
→ 60°
1140° → resto 60°
→ 60°
−400° → −400 + 360 = −40
−40 + 360 = 320°
→ 320°
15π/2 → divide por 2π
15/4 = 3 resto 3π/2
→ 3π/2
(= 270°)
Para radianos: use 2π em vez de 360°
Objetivo: sempre trazer o ângulo para uma volta só
Erro clássico: parar antes de entrar no intervalo correto
Dica: se for negativo, continue somando 360° até ficar positivo
